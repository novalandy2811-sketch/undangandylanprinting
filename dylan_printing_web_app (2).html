Petunjuk Integrasi Penuh Dengan Google Apps Script & Sheets

Aplikasi ini dirancang untuk dapat terhubung langsung ke **Google Spreadsheet** sebagai databasenya. Ikuti 3 langkah mudah berikut untuk mengunggah berkas:

Berkas Backend: Code.gs

Silakan salin seluruh kode dari kotak file editor di sebelah kanan (file Code.gs) dan letakkan di editor proyek Google Apps Script Anda.

Langkah Penyebaran (Deployment):

  1. Buka script.google.com dan buat Proyek Baru.
  2. Salin kode dari Code.gs ke berkas bawaan Google Apps Script.
  3. Buat file HTML baru di proyek tersebut, beri nama persis Index.
  4. Salin semua isi file index.html (aplikasi utama ini) dan tempelkan ke file HTML baru tersebut.
  5. Klik tombol Deploy → New deployment → Pilih jenis Web App.
  6. Pilih opsi: Execute as: "Me" dan Who has access: "Anyone".
  7. Klik Deploy, lalu berikan otorisasi izin akun Google Anda.
